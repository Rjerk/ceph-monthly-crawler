# ceph-monthly-crawler

```
pip3 install beautifulsoup4 selenium xlsxwriter
npm install chromedriver -g

# 查看 chrome://settings/help 获得版本
# 从 http://chromedriver.storage.googleapis.com/index.html 下载相应版本 chrome driver
wget https://chromedriver.storage.googleapis.com/83.0.4103.14/chromedriver_mac64.zip
unzip chromedriver_mac64.zip

sudo mv ~/Downloads/chromedriver /usr/bin

./crawl.py
```
